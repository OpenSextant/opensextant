These jars provide the non-exact string matching algorithms used when creating the place name matching index.  
opensextant-phonetics.jar is built in the OpenSextantToolbox project. The common_codecs.x-x.jar comes from the 
Apache Commons Codec project (http://commons.apache.org/codec/)
These jars should be placed in <KETTLE_HOME>/libext before running the Kettle scripts
